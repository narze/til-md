---
title: console.table()
date: 2020-09-07
tags: javascript
---

Use `console.table()` instead of `console.log()` with object to print it within a table

```javascript
console.table({ width: 1920, height: 1080 })
```

Output :

| (index) | Value |
|---------|-------|
| width   | 1920  |
| height  | 1080  |
